# DotNet FSE Hands-On Labs

## Overview

This repository contains all mandatory hands-on exercises completed as part of the **.NET Full Stack Engineer (FSE) Learning Program**. The repository covers fundamental and advanced concepts across backend development, frontend development, databases, software design principles, testing frameworks, version control, and microservices architecture.

---

### Design Patterns

* Singleton Pattern
* Factory Method Pattern

### Data Structures & Algorithms

* E-Commerce Search Optimization
* Financial Forecasting Algorithms

### Advanced SQL

* Ranking and Window Functions
* Stored Procedures
* User Defined Functions

### NUnit & Moq

* Unit Testing with NUnit
* Mocking Dependencies using Moq

### Entity Framework Core

* ORM Fundamentals
* DbContext Configuration
* Migrations
* CRUD Operations
* LINQ Queries

### ASP.NET Core Web API

* RESTful API Development
* Controllers and Routing
* Dependency Injection
* Exception Handling
* API Documentation

### Microservices

* JWT Authentication
* Service-Based Architecture

### Frontend Development

* ReactJS Hands-On Labs
* Angular Hands-On Labs

### Version Control

* Git Fundamentals
* Branching and Merging
* Repository Management
